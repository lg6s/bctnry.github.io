The rules

"The Game" or this game is a variant of this game. The rules are as follows:

Rule 1: Everybody in the world who knows about The Game is playing The Game.
So if:
1. It's the first time that you've heard of this game, or
2. You've opened this webpage but you refused to learn about it (which is really smart) before,
visiting it didn't make you lose. It's currently one of the only two ways to keep yourself from losing.
(The other way is to declare your leaving explicitly.)

Rule 2: Whenever you thinks about The Game after your first oblivion about it, you lose.
So you didn't really lose it just now.

Rule 3: Losses must be announced.
You can choose any method that is suitable for you.

Rule 4: The game restarts automatically after you've forgotten about it.
So do not ask yourself questions like "when will the game starts again" as it will make you lose immediately.

Rule 5: You can only quit the game by announcing that you quit.
Once you've announced that you have quitted from the game, rules above don't have any effect on you anymore.
That is, thinking about this game don't make you lose anymore.
in addition, when you quit, YOU HAD WON THE GAME.

Rule 6: You must announce your participation if you want to play the game again after quitting it.
It is regarded as a loss; so you need to follow Rule 3 immediately.

Now you have known the rules.
