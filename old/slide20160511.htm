Paradigms

What is paradigm

“programming styles enforced by languages”

  • imperative
    • procedural (e.g. C, Pascal, etc.)
    • object-oriented (e.g. C++, Smalltalk, ruby, etc.)
    • etc.
  • declarative
    • logic (e.g. prolog, miniKanren, etc.)
    • functional (e.g. haskell, lisp, erlang, ML family)
  • others
    • metaprogramming
    • parallel computing & concurrent
      • actor model, etc
    • and others.
  • There's no such things as “interpreted language” or “compiled language”!

Imperative

based on changes of mutable states (i.e. side-effects): 

  int x = 0;
  // this will output 0
  std::cout << x << std::endl;
  // this changes the "state" of x
  x = 3;
  // and now it will output 3
  std::cout << x << std::endl;

“destructive change”; have to care about side-effects.

  • side-effects considered harmful 
    int x = 10;
printf("%i %i %i", x++, ++x, (x++)+(++x));
// what will the line above print on the screen?
  • data race in concurrent scensarioes, etc.

  data Tree a = Nil | Branch a (Tree a) (Tree a) deriving (Eq,Show)
 
  nlr :: Tree a -> [a]
  nlr Nil = Nil
  nlr (Branch x y z) = x:(nlr y ++ nlr z)
 
  nlr' :: Tree a -> [a] 
  nlr' x = case x of { Nil -> [] ; Branch a b c -> a:(nlr' b ++ nlr' c) }

How to implement a language?

Most implementations can be divided into 2 parts:

Parsers & Code Generators

Or “front end” & “back end”

Parser

  • source -> “abstract syntax tree”
  • mainly consist of 2 parts:
    • tokenizer :: string -> list of string
    • parser :: list of string -> abstract syntax tree (or other representations)

Code generator

  • abstract syntax tree -> executable codes
    • real machine codes (ARM, x86, MIPS, etc.)

    • or virtual machine codes

      (e.g. clojure/scala -> Java VM, elixir -> Erlang VM)
    • or some Intermediate Representation
      • LLVM IR
    • … or just another programming language
      • assembly languages, brainfuck(?)

…or you can choose other ways

  • Just-In-Time compiler
  • REPL (Read-Eval-Print-Loop)
  • As a scripting language
